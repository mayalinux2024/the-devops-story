# Chapter 1 — The Empty Land

## Goal

Understand why cloud computing exists before learning any DevOps tool.

---

# 📖 Story

Once upon a time, a king wanted to build the world's greatest toy factory.

He had workers.

He had builders.

He had ideas.

But there was one problem.

He owned no land.

Without land, nothing could be built.

So instead of buying an entire country, the king rented a small piece of land.

Now he could start building immediately.

That rented land became the home of every workshop, road, warehouse, and factory that would come later.

The kingdom could finally grow.

---

# ❓The Problem

Before cloud computing, companies had to buy their own physical servers.

That meant:

- Buying expensive hardware
- Waiting weeks or months for delivery
- Building server rooms
- Installing networking equipment
- Maintaining everything themselves

Even launching a small application could take a long time.

---

# 🌍 Real-World Translation

| Story | Real World |
|---------|------------|
| King | Company |
| Empty Land | Cloud Provider |
| Kingdom | Cloud Infrastructure |
| Toy Factory | Application |

In real life, that rented land is called **AWS**.

Instead of buying physical hardware, companies rent computing resources whenever they need them.

---

# ☁️ Technical Explanation

AWS (Amazon Web Services) is a cloud platform.

It allows businesses to rent infrastructure on demand instead of owning physical servers.

You pay only for the resources you use.

This makes building and deploying applications much faster.

---

# 💻 Mini Lab (5 minutes)

## Goal

Explore the AWS Console.

1. Log in to your AWS account.
2. Notice the Region selector at the top-right.
3. Open **EC2**.
4. Open **S3**.
5. Open **VPC**.

Don't create anything yet.

Simply observe that AWS provides many services, all built on the same cloud platform.
