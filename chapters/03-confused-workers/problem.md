# Story Problem

Inside every workshop...
Workers arrive.

But they ask:

"What do we build?"

Nobody knows.

There are no instructions.

So every morning...

Someone writes instructions on the whiteboard.

Today's Job

1. Make robot
2. Paint blue
3. Put in box

Every workshop writes the instructions again.

Very boring.

Very slow.

# Problem B — Real World
Imagine two Ubuntu servers.

Server A

Python 3.12

Flask installed

Application works.

---

Server B

Python 3.10

Flask missing

Application fails.

---

Both are Linux servers.

Both are EC2 instances.

But they are NOT identical.

This is one of the problems Docker was created to solve.
